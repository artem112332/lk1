Точка сбора

Отправлено
Компания проводит стажировки и производственные практики для студентов в формате проектной школы (аналог проектного практикума). В настоящий момент разрабатывается веб-сервис, содержащий в себе функционал CRM, трекера задач и оценки компетенций. Для этого сервиса нужно разработать прототип Личного кабинета, в котором пользователь - стажёр, куратор, консультант - будет видеть всю необходимую для работы информацию.
Уровни результата (минимальный / базовый / оптимальный):
  • Минимальный: Прототип с базовыми страницами для каждой роли.
  • Базовый: Добавление функциональности редактирования личных данных и отображения ключевой информации.
  • Оптимальный: Полностью функциональный прототип с интеграцией всех элементов (анкетные данные, компетенции, проектные команды, новости, напоминания).

Цель
Прототип Личного кабинета пользователя “Точки сбора”, отображающий значимую информацию о работе в проектах в соответствии с текущей ролью пользователя
Требуемый результат
Рабочий прототип страницы Личного кабинета с возможностью редактировать личные данные и отображающий значимую информацию о работе на проектах:
  • анкетные данные;
  • компетенции;
  • профиль развития профессионально-важных качеств;
  • проектная команда для проектанта, курируемые команды для куратора;
  • список реализованных проектов на стажировках компании;
  • новости о новых стажировках;
  • напоминания о назначенных задачах и т.п.;
  • статус поданных заявок на стажировку и результаты выполненных тестовых заданий.
Критерии оценки
  • прототипы страниц для ролей ИС;
  • оформлена функциональная спецификация;
